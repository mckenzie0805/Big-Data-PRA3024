## Course Overview

Modern physics relies heavily on large and complex datasets generated by experiments and observations.  
This course introduces the systematic analysis of such data, combining programming, statistics, and physical reasoning to extract meaningful results and assess their significance.

Key themes include:
- Data handling and preprocessing
- Statistical analysis and uncertainty estimation
- Error analysis, correlations, and significance testing
- Awareness of false positives in data
- Working with diverse physics datasets
